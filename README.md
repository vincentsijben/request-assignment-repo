# Request Assignment Repo

This repository helps students request their assignment repository in one place.

## For students

1. Open the issue form link shared in Moodle.
2. Fill in the fields.
3. Submit once and wait a few seconds (up to about one minute).
4. Read the bot reply in your issue for the result.

If your request is rejected, follow the retry instructions in that issue comment and submit again.

## What this automation does

1. Validates the request.
2. Creates the target repository from a starter template when needed.
3. Grants the student maintain access.
4. Posts a clear success or rejection comment on the issue.
